    예방 수칙
    1. 링크 클릭 주의

      출처가 미확인 문자의 링크 주소 클릭 주의

    2. 스마트폰 보안 설정 강화

      알 수 없는 출처의 앱 설치 제한

    3. 소액 결제 차단, 제한

      자신의 스마트폰으로 114를 눌러 상담원과 연결

    4. 금융 정보 입력 제한

      보안 승급 명목으로 요구ㅏ는 보안 카드번호 입력 금지

    5. 피싱 의심, 전화 문자를 받은 경우

      인터넷 검색창에 '인터넷 보호나라' 검색 -> 휴대폰 번호 입력하여 확인
